### 综合技能系统

#### [everything-claude-code (ECC)](https://github.com/affaan-m/everything-claude-code)

> 🧠 Agent Harness 性能优化系统：Skills + 记忆 + 安全 + 研究驱动开发

| 项目 | 说明 |
|------|------|
| ⭐ Star | 211.9K+ |
| 🍴 Fork | 32.5K+ |
| 👥 贡献者 | 230+ |
| 📦 类型 | 综合 Agent 操作系统 |
| 🎯 适用 | Claude Code / Codex / Cursor / OpenCode / Gemini / Zed / Copilot |
| 📄 License | MIT |

**核心特点：**
- **261 个 Skills**：覆盖 TypeScript、Python、Go、Java、Kotlin、Rust 等 12+ 语言生态
- **66 个 Agents**：预构建的专业化 Agent，可直接使用
- **记忆系统**：Hooks 自动保存/加载上下文，跨会话持久化
- **安全扫描**：内置 AgentShield，攻击向量检测与沙箱隔离
- **研究驱动开发**：基于调研的代码生成，减少幻觉
- **跨工具支持**：不仅限于 Claude Code，支持多个 AI Agent 工具

**安装方式：**
```bash
# 方式一：通过 npm（推荐）
npm install -g ecc-universal

# 方式二：通过 git
git clone https://github.com/affaan-m/ECC.git
cd ECC
# 参考安装指南进行配置
```
